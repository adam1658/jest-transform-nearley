# jest-transform-nearley

A jest transform that enables nearley grammar support

## Usage

```bash
npm install jest-transform-nearley
```

```js
const grammar = require('./grammar.ne');
```

## Development

Testing:

```bash
npm run test
```

Formatting with prettier:

```bash
npm run format
```
